# Laravel Marketplace

Платформа для размещения заказов и услуг. Заказчики и исполнители находят друг друга.

## 🚀 Быстрый старт

### 1️⃣ При установке или pull

```bash
npm run new
```

Устанавливает зависимости, создаёт .env, базу данных, запускает миграции и сидеры.

### 2️⃣ Запуск проекта

```bash
npm run start
```

Запускает локальный сервер на http://127.0.0.1:8000

### 3️⃣ Подготовка к отправке в репозиторий

```bash
npm run prepare
```

Очищает кэш, сбрасывает .env к дефолтному состоянию, удаляет логи. После выполнения:

```bash
git add . && git commit -m "изменения" && git push
```
